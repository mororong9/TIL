#### 데이터 구조 알고리즘

- 구조를 안다는것: 어떻게 저장하고 어떻게 활용할 수 있는지

- 시퀀스형(조작 가능): 리스트, 튜플, 레인지

- 비시퀀스형:  세트, 딕셔너리

- 순서가있다= 순회가있다=이터러블=정렬가능=인덱스가능



#### 출력

end, sep옵션 사용

```python
a='ha'
b='ppy'
print(a, end='@')
print(b)
#ha@ppy
print(a,b,sep='\n')
#ha
#ppy

a,b,c= map(int, input().split())
#input=1 2 3
print(a,b,c, end='&')
#1 2 3& (들였을때 띄웠으니까 나올때도  띄움)
```
